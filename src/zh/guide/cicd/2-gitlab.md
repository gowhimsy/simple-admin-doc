## Gitlab CI 文件生成

> goctls >= 1.6.6

## 创建 Gitlab CI 文件

> 创建 .gitlab-ci.yaml

```shell
goctls cicd gitlab
```

## 参数介绍

```shell
$ goctls cicd gitlab -h
生成 gitlab-ci.yml 文件

Usage:
  goctls cicd gitlab [flags]

Flags:
  -h, --help                help for gitlab
  -o, --output_dir string   输出目录 (default ".")
```
