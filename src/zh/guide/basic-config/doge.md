## 介绍

Doge 是 Simple Admin 的模块下载部署的命令行工具，提供模块源码下载，模块 docker , k8s 部署，服务器维护等功能。用户可以上传自己的付费模块获取收益，现已收录 10 + 模块。


::: warning
Doge 是一个会员专属工具，由于该工具需要连接服务器下载文件和付费模块，所以只能会员使用，只需加入星球即可成为会员，支持下我们吧😊

:::
